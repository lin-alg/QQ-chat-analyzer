# QQ 聊天记录深度分析器 v2.0

这是一个离线运行的网页工具，可以帮你分析 QQ 聊天记录。

## 使用方法
1. 先用 [qq-chat-exporter](https://github.com/shuakami/qq-chat-exporter) 导出聊天记录（格式为 `.json` 文件）。
2. 打开本工具的网站页面: [QQ 聊天记录深度分析器](https://lin-alg.github.io/QQ-chat-analyzer/)。
3. 上传之前导出的 JSON 文件。
4. 点击 **开始分析**，即可看到聊天统计、活跃时间、热词排行、表情包排行等结果。

## 特点
- **简单易用**：只需要一个导出的 JSON 文件。
- **安全可靠**：所有数据只在你的浏览器里处理，不会上传。
- **处理迅速**: 立即可以看到结果。
